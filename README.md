# Dispense-calogero
Trascrizione in latex delle dispense di Metodi e modelli matematici per la fisica del prof F. Calogero

sono state create con l'ausilio del modello Claude Sonnet 4.6 di Anthropic a partire dalle scansione 
delle dispense originali. lo scopo era quello di trascriverle nel modo piu fedele possibile all'originale.
Ad ora sono sicuramente presenti errori e/o sviste di revisione quindi non è garantita la correttezza del contenuto.
inoltre, il codice sorgente è abbastanza disordinato e commentato in modo disomogeneo.
